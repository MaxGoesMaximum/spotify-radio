# Changelog

All notable changes to Spotify Radio will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/),
and this project adheres to [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2025-01-01

### Added
- 10 FM radio stations (Pop, Rock, R&B, Jazz, Hip-Hop, Dutch, Classics, Indie, Dance, Chill)
- AI DJ announcer with text-to-speech (3 Dutch voices)
- Spotify OAuth authentication with automatic token refresh
- Real-time weather integration (OpenWeatherMap)
- Live news ticker (GNews API)
- Song progress bar with seek functionality
- Cookie consent banner (GDPR)
- Keyboard shortcuts (Space, N, arrows, M, F, ?)
- Progressive Web App (installable, offline support)
- GDPR data export and account deletion
- Fullscreen player mode
- Song reactions (6 emoji types)
- Genre statistics with pie chart
- Listening streak tracking
- Internationalization (Dutch, English, German)
- Holiday-themed DJ announcements
- Sleep timer with volume fade-out
- Invite/share system with station presets
- Retry logic with exponential backoff
- Audit logging for security events
- Language selector in navigation
- Top tracks display
- Audio visualizer (3 styles: bars, wave, circular)
- Light/dark theme support (5 themes)
- Per-station color customization
- Turso cloud database support for Vercel deployment
